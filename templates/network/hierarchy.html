LCP Details

  • • Main distribution point for fiber network
  • • Usually located in central/accessible location
  • • Contains multiple splitters inside cabinet
  • • Serves entire barangay or area
  • • Fed by main fiber trunk from central office

Splitter Types

Type Ports Use Case
1:4 4 Rural/Low density
1:8 8 Standard residential
1:16 16 Medium density
1:32 32 High density/Urban
1:64 64 Very high density

NAP Functions

  • • Secondary distribution point
  • • Located near customer clusters
  • • Weatherproof box on pole/wall
  • • Direct fiber connections to customers
  • • Multiple NAPs can share splitter port (cascading)

Port Management

  • • Green = Available port
  • • Red = Occupied port
  • • 1 Port = 1 Customer premise
  • • Port serves entire location (all devices)
  • • Track usage for capacity planning
