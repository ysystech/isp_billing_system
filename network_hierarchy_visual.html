Understanding the Hierarchy

1. LCP (Local Convergence Point)

  • Main distribution point for an area (e.g., a barangay)
  • Contains multiple splitters
  • Typically covers 1-2km radius

2. Splitters

  • Divides the optical signal from LCP
  • Common types: 1:4, 1:8, 1:16, 1:32, 1:64
  • Each output port connects to a NAP

3. NAP (Network Access Point)

  • Secondary distribution box closer to customers
  • Connected to ONE splitter port
  • Has multiple customer ports (typically 4, 8, or 16)
  • Multiple NAPs can share same splitter port (cascading)

4. Customer Ports

  • Each port = one customer connection
  • Green = available, Red = occupied
  • One port serves one location (house/business)

Capacity Example

If LCP-001 has:

  • 3 Splitters (1:8, 1:16, 1:4) = 28 total splitter ports
  • If each splitter port connects to a NAP with 8 customer ports
  • Total capacity = 28 × 8 = 224 customers

Real-World Scenario

Customer Installation Process:

  1. Customer at "123 Main St" requests connection
  2. Technician identifies nearest NAP (e.g., NAP-001)
  3. Checks available ports (P1, P4, P6, P7 available)
  4. Runs fiber cable from customer to NAP
  5. Connects to available port (e.g., P4)
  6. Updates system: Port P4 now occupied
